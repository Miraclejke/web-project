# GymHelper

Мини-трекер для тренажерного зала и питания. Данные хранятся в `localStorage`.

## Запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:5173`.

## Что внутри

- План тренировок по дням недели (упражнение + комментарий)
- Тренировка по дате с сетами (вес/повторы)
- Дневник питания по дате с подсчетом макросов
- Отдых/сон по дате
- Дашборд со статусом сегодняшней даты
