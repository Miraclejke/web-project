# GymHelper

## Мини-трекер для тренажерного зала, питания и сна.

## Функционал
1) Во вкладке "План" можно состаить расписание своих тренировок на неделю, добавляйте Упражнение из списка или вписывайте сами + оставляйте комментарий.
2) Во вкладке тренировки можно выбрать дату и создать тренировку, куда вы добаляете упражнение и для каждого упражнения есть возможность добавить кол-во поторейний(сетов) и кол-во подходов.
3) Во вкладке питания можно выбрать дату и отмечасть свои приёмы пищи, у каждого приёма пищи можно указывать название блюда и КБЖУ.
4) Во вкладке мон можно выбрать дату и ввести сколько был сон в часах и оставить комментарий по необходимости.
5) Во вкаладке дашбоард можно перейти к вкладкам тренировка, питание, сон на сегодняший день.
6) Во вкаладке дашбоард также можно увидеть статистику за последнии 2 недели своих показателей (кол-во тренировок, средний калораж и кол-во сна в среднем за эти 14 дней).
